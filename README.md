# AI Smart CustomerCare : Real-Time Customer Support Dashboard

An AI-powered full-stack customer support system designed to assist agents with intelligent reply suggestions and real-time chat management.

## 🚀 Project Overview

AI Smart CustomerCare is a scalable customer support platform that enables:

- Secure JWT-based authentication
- Role-based access (Customer / Agent)
- Persistent conversation storage
- Real-time messaging (WebSocket – upcoming)
- AI-based intent detection & smart reply suggestions (upcoming)

This project demonstrates full-stack development, authentication architecture, real-time system design, and applied NLP integration.

---

## 🏗 Architecture

Frontend (React) → FastAPI Backend → MongoDB  
                               ↓  
                          AI Intent Engine  

---

## 🛠 Tech Stack

### Backend
- FastAPI
- MongoDB
- Motor (Async MongoDB Driver)
- JWT Authentication (python-jose)
- Passlib (bcrypt hashing)

### Frontend
- React (Vite)
- Context API
- Tailwind CSS (planned)

### AI (Planned)
- TF-IDF Vectorization
- Naive Bayes Classifier
- Smart Reply Mapping Engine
